# 학습의 갈피를 잡지 못하다

## JS와 TS

JavaScript와 TypeScript는 고수준 언어로, 다음과 같은 특별하고 편리한 문법을 제공합니다:

1. **비구조화 할당 (Destructuring)**
   - 배열이나 객체의 요소를 변수에 쉽게 할당할 수 있습니다.
   ```js
   const { name, age } = { name: "Alice", age: 30 };
   ```

2. **템플릿 리터럴 (Template Literals)**
   - 문자열 내에서 변수를 쉽게 사용할 수 있습니다.
   ```js
   const greeting = `Hello, ${name}`;
   ```

3. **Spread / Rest 연산자**
   - 배열이나 객체를 쉽게 확장하거나 복사할 수 있습니다.
   ```js
   const arr = [1, 2, 3];
   const newArr = [...arr, 4, 5];
   ```

4. **옵셔널 체이닝 (Optional Chaining)**
   - 객체의 중첩된 속성에 안전하게 접근할 수 있습니다.
   ```js
   const value = obj?.nested?.property;
   ```

5. **Nullish Coalescing**
   - 값이 `null`이나 `undefined`일 경우 대체값을 지정할 수 있습니다.
   ```js
   const value = input ?? "default";
   ```

6. **Async/Await**
   - 비동기 코드를 동기식으로 작성할 수 있어 가독성이 향상됩니다.
   ```js
   async function fetchData() {
     const data = await fetch(URL);
   }
   ```

7. **TypeScript의 타입 추론과 제네릭**
   - TypeScript는 자동 타입 추론과 함께, 제네릭을 사용하여 타입을 더 유연하게 처리할 수 있습니다.
   ```ts
   function identity<T>(arg: T): T {
     return arg;
   }
   ```

이러한 문법들은 코드를 간결하게 작성하고 의도를 명확히 표현할 수 있게 도와줍니다.
