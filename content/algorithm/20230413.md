# TIL of Algorithm

## 문제 
두 정수 A와 B를 입력 받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 입력 
첫째 줄에 테스트 케이스의 개수 T가 주어진다. 
각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다(0 < A, B < 10)

## 출력
각 테스트 케이스마다 A+B 를 출력한다. 

## 예제 입력
```shell
5
1 1
2 3
3 4 
9 8
5 2
```

## 예제 출력 
```shell
2
5
7
17
7
```

## 작성 코드 
```python
# https://www.acmicpc.net/problem/10950
cnt = int(input())
for i in range(cnt):
  a, b = map(int, input().split())
print(a+b)

# 최적화 버전
import sys
t = int(input())
for i in range(t):
  a, b = map(int, input().split())
print(a+b)
```
